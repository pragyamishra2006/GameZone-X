# 🎮 GameZone-X

GameZone-X is a modern multi-game web platform built using HTML, CSS, JavaScript and MongoDB. It provides a clean gaming dashboard where users can log in, explore games, track scores, and enjoy a smooth UI experience.

## ✨ Features

- 🔐 Simple login system using localStorage
- 👤 User profile with initials avatar
- 🏆 Leaderboard page for tracking scores
- 🎮 Multiple mini-games in one platform
- 🎯 Clean and modern gaming dashboard UI
- 📱 Fully responsive design
- 🌙 Dark-themed aesthetic UI
- ⚡ Smooth animations and hover effects

---

## 🎮 Games Included

### Classic Games
- 🐍 Snake Game – Grow the snake by eating food
- ⭕ Tic Tac Toe – Play against AI or friend
- 🧠 Memory Game – Match cards and test memory

### Adventure Games
- 🏃 Runner Game – Jump over obstacles
- 🐦 Flappy Bird – Avoid pipes and fly
- 🧱 Breakout – Break all bricks using paddle

### Puzzle Games
- 🔢 2048 – Merge tiles to reach 2048
- ➗ Math Quiz – Solve math problems quickly
- ⚡ Reaction Test – Measure your reaction speed

---

## 🛠️ Tech Stack

- HTML5
- CSS3 (Glassmorphism UI)
- JavaScript (Vanilla JS)
- LocalStorage (for user session)
